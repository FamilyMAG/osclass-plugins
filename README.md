Bitcoins plugins
================

Display the price in Bitcoins currency. It will update with prices hourly




Usage
=====

Add this code wherever you want the price in Bitcoins to appear:

 <?php bitcoins_price(); ?>





Currently supported currencies (to make the conversion from) : 
============
- AUD
- BRL
- CAD
- CHF
- CNY
- DKK
- EUR
- GBP
- HKD
- JPY
- NZD
- PLN
- RUB
- SEK
- SGD
- SLL
- THB
- USD
